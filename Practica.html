HTML: Lenguaje d'etiquetes d'hipertext

Què és l'HTML i per a què serveix?

HTML(Lenguaje de Marques de Hipertexto, del inglés HyperText Markup Language ) és el components més bàsics de la Web. Definiu el significat i l'estructura del contingut web. A més d'HTML, generalment s'utilitzen altres tecnologies per a descriure l'apariència/presentación d'una pàgina web (CSS) o la funcionalitat/comportament (JavaScript).

Etiquetes HTML bàsiques

  • HTML: Defineix l'inici del document HTML, indica al navegador que el que ve a continuació ha de ser interpretat com a codi HTML. Això és així de facto , ja que en teoria el que defineix el tipus de document és el DOCTYPE , que significa la paraula just després de DOCTYPE el tag d'arrel.
  • SCRIPT: Incrusta un script en una web, o truca a un mitjançant src="url del script" . Es recomana incloure el tipus MIME a l'atribut type, en el cas de JavaScript text/javascript.
  • HEAD: Defineix la capçalera del document HTML; aquesta capçalera sol contenir informació sobre el document que no es mostra directament a lusuari com, per exemple, el títol de la finestra del navegador.
  • BODY: Defineix el contingut principal o cos del document. Aquesta és la part del document html que es mostra al navegador; dins d'aquesta etiqueta es poden definir propietats comunes a tota la pàgina, com a color de fons i marges.

Tipos de lenguaje

HTML CSS JavaScript
se encarga de desarrollar una descripción sobre los contenidos que aparecen como textos y sobre su estructura, complementando dicho texto con diversos objetos (como fotografías, animaciones, etc). El diseño del CSS posibilita establecer una separación entre el contenido y la forma de presentación del documento (dada por las fuentes, los colores y las capas empleadas). JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, cada vez que una página web hace algo más que sentarse allí y mostrar información estática para que la veas, muestra oportunas actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo, etc.
